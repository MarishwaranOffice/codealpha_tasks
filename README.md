# CodeAlpha Python Internship Tasks

This repository contains my CodeAlpha Python Internship Projects, completed following the official internship instructions. It includes three professional-level applications demonstrating Python programming, GUI development, automation, and problem-solving skills.

## Projects Included

**Hangman Game**  
- A complete GUI-based Hangman game using Tkinter.  
- Features:  
  - Random word selection from a predefined list  
  - Letter-guessing interface with clickable buttons  
  - Lives/tries counter with hangman drawing  
  - Countdown timer for extra challenge  
  - Win/Lose detection with messages  
  - Restart functionality  
- Purpose: Demonstrates Python loops, conditionals, GUI, and event handling  

**Stock Portfolio Tracker**  
- A professional stock tracker application using Tkinter.  
- Features:  
  - Add multiple stocks with quantity  
  - Predefined hardcoded stock prices  
  - Calculates per-stock value and total investment  
  - Save portfolio to CSV file  
  - Clear portfolio and reset functionality  
  - Input validations with error messages  
- Purpose: Demonstrates Python dictionaries, file handling, GUI design, and calculations  

**Task Automation Tool**  
- A versatile automation application.  
- Features:  
  - Move all `.jpg` files from one folder to another  
  - Extract emails from a `.txt` file and save to another file  
  - Scrape webpage title and save to a file  
  - Dynamic GUI fields based on task selection  
  - Logs actions with timestamps  
  - Clear fields and Exit buttons  
  - Robust error handling  
- Purpose: Demonstrates Python os, shutil, re, requests, file handling, GUI, and automation

## How to Run
- Clone the repository:  
`git clone https://github.com/YOUR_GITHUB_USERNAME/codealpha_tasks.git`  
- Navigate into the repository folder:  
`cd codealpha_tasks`  
- Run any Python file using:  
`python CodeAlpha_HangmanGame.py`  
`python CodeAlpha_StockPortfolio.py`  
`python CodeAlpha_TaskAutomation.py`  

Make sure Python 3.x is installed. Tkinter comes pre-installed with standard Python distributions. For Task Automation, `requests` library is required:  
`pip install requests`
